What we offer // Oferta

We are a group of enthusiasts of generally understood technology, programming and electronics. We always try to follow the latest solutions, reducing the technological debt as much as possible. We deal with programming and electronics professionally and as a hobby. // Jesteśmy grupą pasjonatów ogólnie pojętej technologii, programowania i elektroniki. Zawsze staramy się podążać za najnowszymi rozwiązaniami, maksymalnie zmniejszając dług technologiczny. Zawodowo i hobbystycznie zajmujemy się programowaniem i elektroniką.

Firmware // Oprogramowanie

We create firmware for microcontrollers based on Cortex cores and others, we adapt to the selected technology. // Tworzymy oprogramowanie układowe dla mikrokontrolerów opartych o rdzenie Cortex i inne, dostosowujemy się do wybranej technologii.

Hardware // Sprzęt

We are able to offer you a ready-made hardware solutions as well as consulting. We create schematic diagrams and PCB designs. // Jesteśmy w stanie zaproponować gotowe rozwiązania sprzętowe, jak również doradztwo. Tworzymy schematy ideowe oraz projekty PCB.

System architecture // Architektura

Optionally, we prepare a full system architecture for the projects we are working on. // W razie potrzeby przygotowujemy pełną architekturę systemową do projektów, nad którymi pracujemy.

Documentation // Dokumentacja

We can also create a readable documentation. // Możemy również stworzyć czytelną dokumentację.

Consulting // Doradztwo

Do you have doubts? Are you stuck somewhere in your project? We will be happy to advise you. // Masz wątpliwości? Utknąłeś gdzieś w swoim projekcie? Z chęcią Ci doradzimy.
